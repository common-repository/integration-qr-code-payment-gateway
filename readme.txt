== Description ==

This Plugin enables WooCommerce shopowners to instant payments through bank  apps like  banking  app to save payment gateway charges in Mongolia.

== Installation ==

= Requirements =

* A  [negpay Merchant](http://merchant.negpay.mn/) account
* An installed  [Woocommerce](https://wordpress.org/plugins/woocommerce/) plugin


= Plugin installation =

1. Visit 'Plugins > Add New'.
2. Search for 'Negpay QR Code Payment Gateway' and install it.
3. Or you can upload the `negpay-qr-code-payment-for-woocommerce` folder to the `/wp-content/plugins/` directory manually.
4. Activate QR Code Payment Gateway from your Plugins page.
5. After activation go to 'WooCommerce > Settings > Payments > negpay QR Code'.
6. Enable options and save changes.
